║  🎯 DESCRIÇÃO DO PROJETO:                                                 ║
║  Interface React moderna e responsiva para criação e gerenciamento de     ║
║  posts educacionais, desenvolvida com Design System consistente e          ║
║  integração completa com a API REST.                                       ║
║                                                                            ║
║  🏗️ ARQUITETURA DO SISTEMA:                                               ║
║  • Component-Based Architecture com componentes reutilizáveis              ║
║  • Context API para gerenciamento de estado global                         ║
║  • Custom Hooks para lógica reutilizável                                   ║
║  • Service Layer para abstração da comunicação com API                     ║
║  • Design System centralizado com tema consistente                         ║
║                                                                            ║
║  🛠️ TECNOLOGIAS UTILIZADAS:                                               ║
║  • Frontend: React 18 + Vite + React Router DOM                            ║
║  • Estilização: Styled Components + Design System                         ║
║  • Formulários: React Hook Form + Validação customizada                    ║
║  • HTTP Client: Axios com interceptors                                     ║
║  • UI/UX: Lucide React + React Hot Toast                                  ║
║  • Qualidade: ESLint + Prettier + Custom Hooks                            ║
║                                                                            ║
